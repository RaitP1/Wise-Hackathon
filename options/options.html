OpenAI API Configuration

This plugin uses OpenAI's GPT-4 to extract invoice data semantically. You need to provide your own API key.

Your API key is stored securely in your browser and never sent to third parties. Get your API key

How It Works

  1. Open an invoice webpage or PDF in your browser
  2. Click the extension icon to open the sidebar
  3. Click "Extract Invoice Data" to automatically fill the form
  4. Review and edit the extracted data if needed
  5. Click "Send Payment" to process the transfer

Supported Fields

  • Receiver Name
  • IBAN / Account Number
  • Amount & Currency
  • Payment Reference (KID/Viitenumber)
  • Invoice Number
  • Issue Date & Due Date
  • Payment Description

The AI automatically detects these fields in any language and format.

Privacy & Security

  • Invoice data is only processed when you click "Extract Invoice Data"
  • Your API key is stored locally in your browser
  • Invoice content is sent to OpenAI's API for processing
  • No data is stored on external servers (except OpenAI's processing)
  • All extraction happens on-demand, not automatically
